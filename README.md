# 📝 Minimalist To-Do App

A clean, drag-and-drop task manager built with HTML, CSS, and JavaScript. Features include:

- ✅ Task creation and categorization
- 🎨 Priority levels with color coding
- 📦 Local storage persistence
- 📁 Export to Excel
- 🧹 Delete tasks
- 📂 Custom categories
- 📱 Responsive design
